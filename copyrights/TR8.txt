The datasets pignatti.rda is a R version of the spreadsheet file which can be found at
 http://flora.garz.net/documents/Ellensb-dyn.xls; the file is part of the following 
scientific publication:

"Pignatti S., Menegoni P., Pietrosanti S., 2005, Biondicazione attraverso le piante vascolari.
Valori di indicazione secondo Ellenberg (Zeigerwerte) per le specie della Flora di Italia. 
Braun-Blanquetia 39, Camerino, pp.  97."
